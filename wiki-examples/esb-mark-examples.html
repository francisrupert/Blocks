Showing/Hiding Marks

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Illum maiores numquam sed iusto vero, suscipit facilis.

All marks on a page can be shown/hidden by firing custom javascript events. To hide all marks trigger a hide-all-esb-marks event. To show all marks, trigger a show-all-esb-marks event. 
						
//jquery Hide
$(document).trigger("hide-all-esb-marks");

//jquery Show
$(document).trigger("show-all-esb-marks");

//Javascript Hide
if (window.CustomEvent) {
    var event = new CustomEvent('hide-all-esb-marks');
} else {
    var event = document.createEvent('CustomEvent');
    event.initCustomEvent('hide-all-esb-marks', true, true);
}
document.dispatchEvent(event);

//Javascript Show
if (window.CustomEvent) {
    var event = new CustomEvent('show-all-esb-marks');
} else {
    var event = document.createEvent('CustomEvent');
    event.initCustomEvent('show-all-esb-marks', true, true);
}
document.dispatchEvent(event);

Using the visible-on-load="false" option will prevent the Mark from appearing on page load. It can then be made visible using Javascript or by using the keyboard shortcut. This option is best used if applied at the page or project level.

To apply this option at the page level, add data-esb-mark-config data-esb-visible-on-load-"false" to the <body> tag.

Full Markup Example 
						
<html>
    <head>
        <link rel="stylesheet" href="/path/to/esb.min.css"/>
        <script src="/path/to/esb.min.js"></script>
    </head>
    <body data-esb-mark-config data-esb-visible-on-load="false">
        <p data-esb-mark>
            Lorem ipsum dolor sit amet, consectetur adipisicing elit. Illum maiores 
            numquam sed iusto vero, suscipit facilis. Dolorem laudantium, temporibus 
            tenetur tempora! Eos voluptatibus, repellendus rerum cupiditate cum 
            laudantium aliquid dolorem!
        </p>
        
        <fieldset data-esb-mark>
            <h2>Sign up for our newsletter</h2>
            <label>Name</label>
            <input type="text" placeholder="Please enter your name"><br/><br/>
            
            <label>Email</label>
            <input type="email" placeholder="me@example.com">
        </fieldset>
        
        <p data-esb-mark style="display:inline-block;">
            <img src="http://placehold.it/150x150" style="float:right;" data-esb-mark>
            The image within this component is floated right, and Mark respects 
            and maintains that float when the label and outline are added.
        </p>
    </body>
</html>
To apply this option throughout your project, modify your project's config.json file: 
						
{
    "logging":"false",
    ...
    "marks": {
        "visible-on-load": false
    }
}

  • purple
  • pink
  • brown
  • black
  • white (useful on dark backgrounds)

Create Your Own Groups

Any value provided to data-esb-group will be appended with esb-mark-group- and added as a class to the Mark's wrapper. This allows you to create your own groups and styles to match.

Full Markup Example 
						
<html>
    <head>
        <link rel="stylesheet" href="/path/to/esb.min.css"/>
        <link rel="stylesheet" href="css-overrides.css"/>
        <script src="/path/to/esb.min.js"></script>
    </head>
    <body>
        <p data-esb-mark data-esb-group="custom-callout">
            Lorem ipsum dolor sit amet, consectetur adipisicing elit. 
            Illum maiores numquam sed iusto vero, suscipit facilis. 
            Dolorem laudantium, temporibus tenetur tempora! Eos 
            voluptatibus, repellendus rerum cupiditate cum laudantium 
            aliquid dolorem!
        </p>
    </body>
</html>
