ESB Frame Styling Examples

The following examples show the most common use cases for styling. This is by no means a comprehensive guide for what can be done to style a Frame.

Default Style

						
<div data-esb-page-framer="http://example.com"></div>

Basic Style Override (Color, Font, link style, etc.)

html - includes Google Font Reference 
						
<link href='http://fonts.googleapis.com/css?family=Oswald' rel='stylesheet' type='text/css'>
<div data-esb-page-framer="http://example.com"></div>
Additional Styles 
						
.esb-page-framer {
    background: skyblue;
    box-shadow: none;
    border: solid 1px #1f8dba;
    font-family: 'Oswald', sans-serif;
  }

  .esb-page-framer:hover {
    background: #b3e0f2;
  }

  .esb-page-framer-title {
    color: white;
    padding-bottom: 0;
  }

  .esb-page-framer-link:hover {
    text-decoration: none;
  }

  .esb-page-framer-link:hover .esb-page-framer-title {
    text-decoration: underline;
  }

  .esb-page-framer-caption {
    color: #186c8e;
    font-style: italic;
  }

  .esb-page-framer-dimensions-annotation {
    color: #1f8dba;
  }

  .esb-page-framer-dimensions-annotation svg {
    fill: #1f8dba;
  }

  .esb-page-framer-iframe-wrap {
    border-top: solid 1px #1f8dba;
  }

Layout Override - Title & Caption on Bottom

By using Flexbox for layout you can completely reorder the elements within each Frame.

html - Example with link and one without 
					
<div data-esb-page-framer="http://example.com" data-esb-title="example.com" data-esb-caption="Just some filler content"></div>
<div data-esb-page-framer="http://example.com" data-esb-title="example.com" data-esb-caption="Just some filler content" data-esb-href="false"></div>
Additional Styles 
					
  .esb-page-framer-inner-wrap {
    display: flex;
    flex-direction: column;
  }
  
  .esb-page-framer-iframe-wrap {
    order: 1;
  }

  .esb-page-framer-title {
    order: 2;
  }

  .esb-page-framer-caption {
    order: 3;
  }

  .esb-page-framer-dimensions-annotation {
    order: 4;
  }

Layout Example

With flexbox, you can change the order of individual elements to create whatever internal layout you want. For example, title & caption on top and dimensions annotation on bottom:

html - Example with link and one without Additional Styles 
					
  .esb-page-framer-inner-wrap {
    display: flex;
    flex-direction: column;
  }
  
  .esb-page-framer-iframe-wrap {
    order: 3;
  }

  .esb-page-framer-title {
    order: 1;
  }

  .esb-page-framer-caption {
    order: 2;
  }

  .esb-page-framer-dimensions-annotation {
    order: 4;
    padding-top: 10px;
  }

Min-Width Override

By default, the .esb-page-framer class has a min-width of 200px. This ensures each framer has a reasonable width to display the title, caption and dimensions annotation.

html - Two examples, max width override is in styles 
					
<div data-esb-page-framer="http://example.com" data-esb-title="example.com" data-esb-caption="I have a min width of 200px" data-esb-viewport-width="320" data-esb-scale="0.25"></div>
<div data-esb-page-framer="http://example.com" data-esb-title="example.com" data-esb-viewport-width="320" data-esb-scale="0.25" data-esb-dimensions="false"></div>
Additional Styles

Note, the title has been styled to be much smaller so it fits within the width.

					
  .esb-page-framer {
    min-width: 0;
  }

  .esb-page-framer-title {
    font-size: 10px;
    padding: 5px;
  }
