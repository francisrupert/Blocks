ESB Component Frame Examples

The following examples show how Frame can be used in conjunction with Inject.

Loading a Component in a Frame

To frame a component, specify the component name for the data-esb-frame attribute and add a data-esb-variation attribute to specify the variation.

By default components are rendered at 100% scale and the Frame will be sized to fit the component's width and height. 


<div data-esb-frame="newsletter-subscription" 
	 data-esb-variation="default"></div>

Title, Caption, & Dimensions Annotation

Title and Caption elements can be applied by using the data-esb-title and data-esb-caption attributes respectively.

The default dimensions annotation can be disabled by specifying data-esb-dimensions="false" 


<div data-esb-frame="newsletter-subscription" 
	 data-esb-variation="default"
	 data-esb-title="Newsletter Component"
	 data-esb-caption="This is a cool component!"
	 data-esb-dimensions="false"
 ></div>

Frame Links

By default the Frame will be linked to the Component Frame template (essentially a basic html page that renders the component). You can override this behavior by passing in a different url for data-esb-href.

You can disable the linked behavior by specifiying data-esb-href="false". 


<div data-esb-frame="newsletter-subscription" 
     data-esb-variation="default" 
     data-esb-title="Default Link" 
></div>	

<div data-esb-frame="newsletter-subscription" 
     data-esb-variation="default" 
     data-esb-title="Linked to Example.com" 
     data-esb-href="http://example.com" 
></div>	

<div data-esb-frame="newsletter-subscription" 
     data-esb-variation="default" 
     data-esb-title="No Link" 
     data-esb-href="false" 
></div>

Changing the viewport

If you need to demonstrate responsive behavior of a component you can change the viewport of the frame using the data-viewport-width attribute.

In this example the button on the newsletter sign up form goes full width and the header reduces its font size at 320px and below. 


<div data-esb-frame="newsletter-subscription" 
     data-esb-variation="default" 
     data-esb-viewport-width="320" 
></div>

Cropping the Frame

If you're displaying a gallery of components and would like to show several small multiples you can use cropping to show just a relevant portion of each component.

Cropping works by setting data-width and data-height to values smaller than than the component's default size and adding data-esb-crop="true" to the Frame. 


<div data-esb-frame="newsletter-subscription" 
     data-esb-variation="default" 
     data-esb-width="150" 
     data-esb-height="150" 
     data-esb-crop="true" 
></div>

<div data-esb-frame="newsletter-subscription" 
     data-esb-variation="red" 
     data-esb-width="200" 
     data-esb-height="150" 
     data-esb-crop="true" 
></div>

<div data-esb-frame="newsletter-subscription" 
     data-esb-variation="blue" 
     data-esb-width="150" 
     data-esb-height="200" 
     data-esb-crop="true" 
></div>

<div data-esb-frame="newsletter-subscription" 
     data-esb-variation="green" 
     data-esb-width="200" 
     data-esb-height="200" 
     data-esb-crop="true" 
></div>

Scaling the Frame

Components can be displayed at less than 100% scale by setting data-esb-scale to a value less than 1. This is an additional way to create small multiples 


<div data-esb-frame="newsletter-subscription" 
	 data-esb-variation="default" 
	 data-esb-scale="0.5"
></div>

<div data-esb-frame="newsletter-subscription" 
	 data-esb-variation="default" 
	 data-esb-scale="0.25"
></div>

<div data-esb-frame="newsletter-subscription" 
	 data-esb-variation="green" 
	 data-esb-scale="0.25"
></div>

<div data-esb-frame="newsletter-subscription" 
	 data-esb-variation="red" 
	 data-esb-scale="0.25"
></div>

<div data-esb-frame="newsletter-subscription" 
	 data-esb-variation="blue" 
	 data-esb-scale="0.25"
></div>

Frame Offset

You can specify a vertical and horizontal offset for the Frame. If no width and height are specified for the Frame, then the Frame's background (default: gray) will appear in the empty space.

data-esb-offset-x will shift the component left by the number of pixels specified and data-esb-offset-y will shift the component up by the number of pixels specified.

Combining offsets with cropping can create some interesting views of each component. 


<div data-esb-frame="newsletter-subscription" 
     data-esb-variation="default" 
     data-esb-offset-x="10" 
     data-esb-offset-y="10" 
></div>

<div data-esb-frame="newsletter-subscription" 
	 data-esb-variation="default" 
	 data-esb-offset-x="10" 
	 data-esb-offset-y="10" 
	 data-esb-width="150" 
	 data-esb-height="150" 
	 data-esb-crop="true"
 ></div>

<div data-esb-frame="newsletter-subscription" 
	 data-esb-variation="default" 
	 data-esb-offset-x="50" 
	 data-esb-offset-y="50" 
	 data-esb-width="150" 
	 data-esb-height="150" 
	 data-esb-crop="true"
 ></div>

<div data-esb-frame="newsletter-subscription" 
	 data-esb-variation="default" 
	 data-esb-offset-x="50" 
	 data-esb-offset-y="100" 
	 data-esb-width="150" 
	 data-esb-height="150" 
	 data-esb-crop="true"
 ></div>

<div data-esb-frame="newsletter-subscription" 
	 data-esb-variation="default" 
	 data-esb-offset-x="200" 
	 data-esb-offset-y="50" 
	 data-esb-width="150"

 ></div>
