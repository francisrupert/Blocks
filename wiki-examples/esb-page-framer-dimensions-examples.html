ESB Page Framer Dimension Examples

There are several data-esb- attributes that control the overal dimensions of each page framer. These options have quite a bit of overlap, so here are some examples for how to get your Page Framer just the right size.

Recommended Approach

The easiest way to get your Page Framer sized correctly is to think about two parameters: viewport-width and scale. Basically "what size do I want to show this page and how scaled down do I want it to be?"

For example, let's say you want to show your design at 320px (a common mobile size), but you want that 320px design to be scaled down to 50%.

						
<div data-esb-page-framer="http://eightshapes.com" 
     data-esb-viewport-width="320" 
     data-esb-scale="0.5"></div>

Maybe you want to show multiple sizes all scaled down by 33% to highlight a responsive design. Note the hight is automatically calculated assuming an aspect ratio of 1.5

						
<div data-esb-page-framer="http://eightshapes.com" 
     data-esb-viewport-width="320" 
     data-esb-scale="0.33"></div>

<div data-esb-page-framer="http://eightshapes.com" 
     data-esb-viewport-width="768" 
     data-esb-scale="0.33"></div>

<div data-esb-page-framer="http://eightshapes.com" 
     data-esb-viewport-width="1200" 
     data-esb-scale="0.33"></div>

Controlling height

By default Page Framer will automatically calculate height based on an aspect ratio of 1.5. This calculation can be overridden by explicitly providing a data-esb-height attribute. 

						
<div data-esb-page-framer="http://eightshapes.com" 
     data-esb-viewport-width="320" 
     data-esb-scale="0.33"
     data-esb-height="300"
     ></div>

<div data-esb-page-framer="http://eightshapes.com" 
     data-esb-viewport-width="768" 
     data-esb-scale="0.33"
     data-esb-height="300"
     ></div>

<div data-esb-page-framer="http://eightshapes.com" 
     data-esb-viewport-width="1200" 
     data-esb-scale="0.33"
     data-esb-height="300"
     ></div>

Controlling width (auto-scale)

By default Page Framer will automatically calculate width based on the data-esb-viewport-width and the data-esb-scale. If you instead provide data-esb-viewport-width and a data-esb-width attribute, the scale will be calculated automatically. 

						
<div data-esb-page-framer="http://eightshapes.com" data-esb-viewport-width="320" data-esb-width="300"></div>
<div data-esb-page-framer="http://eightshapes.com" data-esb-viewport-width="768" data-esb-width="300"></div>
<div data-esb-page-framer="http://eightshapes.com" data-esb-viewport-width="1200" data-esb-width="300"></div>

Zooming

If the data-esb-viewport-width is smaller than the data-esb-width provided, the content will be zoomed to larger than 100%. 

						
<div data-esb-page-framer="http://eightshapes.com" 
     data-esb-viewport-width="320" 
     data-esb-width="400"></div>

Controlling aspect ratio

Be default Page Framer uses a 1.5 aspect ratio to calculate the height. This aspect ratio can be changed by providing the data-esb-viewport-aspect-ratio attribute. The aspect ratio must be a valid number, i.e. 16:9 won't work, use 0.5625 instead. 

						
<div data-esb-page-framer="http://eightshapes.com" 
     data-esb-viewport-width="320" 
     data-esb-viewport-aspec-ratio="0.5625"></div>
