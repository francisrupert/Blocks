Page Frame

Frame a scaled page layout in a predictable responsive context.

Use To

  • Embed in documentation, avoiding screenshots
  • Display adjacent to notes or specs
  • Compare as small multiples or in device frames
  • Present in gallery or list formats
Example Code Tutorial & Doc 
<div data-esb-frame="site/sign-in-page.html" 
     data-esb-title="Sign In Page" 
     data-esb-height="200"></div>
