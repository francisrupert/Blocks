Navbar

Houses both branding and navigation for the entire site.

Variations

  • Default
  • Brand Only
  • Inverted

Default


Code

Responsive Behavior




States

Brand Only


Use When
Use on lead / e-commerce conversion pages (checkout, form submission). Eliminates distractions and helps boost conversion percentages.

Inverted


Use When
Use when the content on the page would be better served by an alternate navbar background.
